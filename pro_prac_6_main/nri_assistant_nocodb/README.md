# Offline NRI Assistant Database

## Purpose

This folder contains a sample database structure for an offline tabletop role-playing game assistant.
The model focuses on campaign management, game sessions, combat encounters, actions, dice rolls, status effects and notes.

## Contents

- CSV tables with sample records
- PlantUML file with the database model
- supporting description for repository demonstration

## Main entities

- users
- campaigns
- game_sessions
- player_characters
- npcs
- combat_encounters
- combat_participants
- game_actions
- dice_rolls
- status_effects
- inventory_items
- notes
