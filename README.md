# Базы данных NocoDB. Проектный практикум

Репозиторий содержит материалы по проектированию структуры базы данных для проекта, связанного с выпускной квалификационной работой.

В составе репозитория представлены:
- таблицы базы данных в формате CSV с примерами заполнения;
- описание состава данных;
- пояснение по назначению таблиц;
- материалы для последующего импорта в NocoDB.

Основная папка проекта:
- Базы Данных NOCODB НРИ-Помощник
