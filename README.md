# Базы данных NocoDB. Проектный практикум

Репозиторий содержит материалы по проектированию структуры базы данных для проекта, связанного с выпускной квалификационной работой.

В составе репозитория представлены:
- таблицы базы данных в формате CSV;
- описание сущностей и связей между ними;
- схема логической структуры данных;
- краткая инструкция по использованию материалов в NocoDB.
