# Схема данных по дисциплине Проектный практикум и связи

## Основные сущности

- Пользователи
- Кампании
- Игровые сессии
- Персонажи
- NPC
- Боевые столкновения
- Участники боя
- Броски кубов
- Правила
- Заметки
- Предметы инвентаря
- Журнал событий

## Основные связи

- Пользователи.id -> Кампании.game_master_id
- Пользователи.id -> Персонажи.owner_user_id
- Кампании.id -> Игровые_сессии.campaign_id
- Кампании.id -> Персонажи.campaign_id
- Кампании.id -> NPC.campaign_id
- Игровые_сессии.id -> Боевые_столкновения.session_id
- Игровые_сессии.id -> Броски_кубов.session_id
- Игровые_сессии.id -> Журнал_событий.session_id
- Игровые_сессии.id -> Заметки.session_id
- Боевые_столкновения.id -> Участники_боя.encounter_id
- Персонажи.id -> Предметы_инвентаря.character_id

## Комментарий к модели

Данная структура описывает базовые информационные потоки приложения НРИ-Помощник.
Центральной сущностью является игровая кампания, внутри которой ведутся игровые сессии, учет персонажей, NPC, боевые события и журнал действий.
