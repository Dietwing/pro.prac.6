# Базы данных NocoDB для системы НРИ-Помощник

## 1. Назначение работы

В данной работе подготовлена структура базы данных для информационной системы, предназначенной для ведения данных настольно-ролевой игры в локальном формате.

Подготовленная структура ориентирована на хранение и обработку следующих видов информации:
- пользователи системы;
- игровые кампании;
- игровые сессии;
- персонажи игроков;
- неигровые персонажи;
- боевые столкновения;
- участники боя;
- результаты бросков кубов;
- справочные игровые правила;
- заметки по ходу кампании;
- предметы инвентаря;
- журнал игровых событий.

## 2. Состав репозитория

В папке проекта находятся:
- CSV-файлы таблиц базы данных с примерами строк;
- файл README с описанием структуры;
- материалы для дальнейшего импорта таблиц в NocoDB.

## 3. Перечень таблиц базы данных

1. Пользователи.csv  
   Содержит сведения о пользователях приложения и их ролях.

2. Кампании.csv  
   Содержит сведения о кампаниях, игровом мире и статусе кампании.

3. Игровые_сессии.csv  
   Содержит информацию о сессиях внутри кампаний.

4. Персонажи.csv  
   Содержит карточки персонажей игроков и их характеристики.

5. NPC.csv  
   Содержит сведения о неигровых персонажах и противниках.

6. Боевые_столкновения.csv  
   Содержит информацию о боевых сценах в рамках игровых сессий.

7. Участники_боя.csv  
   Содержит состав участников конкретного боя и их текущее состояние.

8. Броски_кубов.csv  
   Содержит журнал игровых бросков кубов.

9. Правила.csv  
   Содержит краткий справочник правил.

10. Заметки.csv  
    Содержит заметки по кампании, сессиям и персонажам.

11. Предметы_инвентаря.csv  
    Содержит перечень предметов, принадлежащих персонажам.

12. Журнал_событий.csv  
    Содержит события, происходившие по ходу игровых сессий.

## 4. Назначение заполнения таблиц

В таблицах приведены абстрактные примеры записей. Эти данные используются для демонстрации структуры базы данных, состава полей и логики хранения информации в рамках предметной области проекта.

## 5. Использование материалов

Подготовленные таблицы могут использоваться:
- при демонстрации структуры базы данных преподавателю;
- при импорте в NocoDB;
- при дальнейшей доработке программной части проекта;
- при построении логических и физических моделей базы данных.
